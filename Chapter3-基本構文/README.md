# 変数
変数とは…  
「データを入れる箱」のようなもの。  
難しく言うと、「メモリ上で管理するデータを保持する領域」。  

# 変数の宣言  
Goは**静的型付け言語**であるため、必ず**変数の宣言**をしなければならない。  
Goの変数の宣言の仕方は複数ある。  

## Varを使った方法
基本形：```var 変数名　データ型```  
ただし、**初期化する際にそのままデータを入れる**ことで、**型推論**をして**データ型を省略することができる**！！  

```go
//「var 変数名 データ型」の例
var n int = 150

//「var 変数名」の例
var lang = "文字列"
```  
また、**まとめて宣言する**ことも可能。  
```go
var (
	num1 = 10
	lang1 = "国語"
)
```

## 関数内のみでしか利用できない変数宣言方法
基本的には上記の変数宣言で問題はないが、  
Goには、**関数内で利用出来る**変数宣言方法がある。  

```go
func main() {
	//Hello関数呼び出し
	hello()
}

func hello(){
	word := "Hello"
	fmt.Println("「関数内で利用できる変数宣言例」：" + word)
}
```
