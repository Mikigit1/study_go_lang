# 演算子  
演算子とは…  
計算や値の比較をする際に利用される記号のこと。  
  
Goの演算子一覧→<https://golang.org/ref/spec#Operators>  
  
# 算術演算子  
中でも特に計算をする為の演算子を**算術演算子**という。  
基本的に他言語と同じなので難しいことはないはず。  
   
|Operator|Description|Example|
|:---|:---|:---|
|+|加算|1 + 2|
|-|減算|2 - 1|
|*|乗算|2 * 1|
|/|除算|2 / 1|
|%|あまり|3 % 1|  
  
+と-は、+100や-100といったように符号としても使うことが可能。
